Description

Originally called the 15-puzzle (also called Gem Puzzle, Boss Puzzle, Game of Fifteen, Mystic Square and many other names), this game is a sliding puzzle that consists of a frame of numbered square tiles in random order with one tile missing. The puzzle also exists in other sizes, particularly the smaller 8-puzzle.

Rules

  • Only one tile can be moved at a time, and it can only be moved into the empty space.
  • Tiles can be moved vertically or horizontally but not diagonally.
